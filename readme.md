# WordCloud

### Build:
- Have CMake installed
```
git clone --recursive https://github.com/recidivism5/WordCloud.git
cd WordCloud
mkdir build
cd build
cmake ..
make //If using gnu tools. Otherwise open the generated solution in Visual Studio.
```
